Probabilistic Topic Modelling¶

Topic Modelling, or Probabilistic Topic Modelling (PTM), is a class of machine learning techniques used to discover the underlying topics in a text document or collection of documents.

The basic assumption behind PTM is that a document can be represented by a set of latent topics, multinomial distributions over words. Each document has then a set of topics and probability distributions associated with it. At the same time, each topic has a set of words and their probabilites of occurrence given that document and topic.

LDA¶

Latent Dirichlet Allocation (LDA) is a type of unsupervised machine learning algorithm that views documents as bags of words (collection of words per document with no regard to word order). LDA assumes a document is generated by picking a set of topics and then for each topic it picks a set of words. So LDA is able to infer the topics in a collection of documents by reverse engineering this assumption in the following way:

  1. Given a k number of topics
  2. Distribute these k topics across document m following a Dirichlet distribution, and assign each word a topic.
  3. For each word w in document m , assign word w a topic

More formally:

  1. Choose N ∼ Poisson(ξ).
  2. Choose θ ∼ Dir(α).

  3. For each of the N words wn:

    a. Choose a topic zn ∼ Multinomial(θ).

    b. Choose a word wn from p(wn |zn,β), a multinomial probability conditioned on the topic zn.

Evaluating Topic Models¶

  1. Perplexity: It measures the log-likelihood of a held-out test set. In other words, it estimates how a set of words not used for training, falls into the topic distribution inferred by LDA.
  2. Coherence: given the n top words in a topic, it measures the pairwise likelihood of two words appearing in a document.

Perplexity is not correlated with human judgments (Newman et al, 2010)
